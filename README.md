# 🏗️ Local AI Office Automator (LAOA)

> **보안과 효율을 동시에 잡는 공사 행정 자동화 솔루션**

## 📌 Project Overview
이 프로젝트는 전기/소방 공사 현장에서 발생하는 방대한 행정 업무(공무)를 로컬 LLM(Ollama)을 활용하여 자동화하는 시스템입니다. 외부 클라우드 AI를 사용하지 않아 기업 비밀 및 관공서 보안 데이터를 안전하게 처리할 수 있습니다.

## 🚀 Key Features
- **Local AI Chat:** 보안 위험 없는 내부 데이터 기반 질의응답 (Ollama - Llama 3)
- **Legal Search (RAG):** 최신 전기설비규정(KEC) 및 소방법규 기반 지식 제공 (예정)
- **Doc Generator:** 공문 및 준공 서류 초안 자동 생성 (예정)

## 🛠 Tech Stack
- **Backend:** Node.js, Express
- **AI Engine:** Ollama (Llama 3)
- **Database:** PostgreSQL (Vector DB 확장 예정)
- **Frontend:** React (Next.js)
